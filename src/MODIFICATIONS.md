# Modifications to Bundled CIANNA Source

The `src/` tree contains the CIANNA C/CUDA backend source used by ORSDet.
It is based on CIANNA by David Cornu, released under the Apache License,
Version 2.0.

This ORSDet distribution may include modifications to the bundled CIANNA source
and build files for repository packaging, local Python extension builds,
CUDA/OpenBLAS configuration, SKAO SDC1 workflow integration, and ORSDet runtime
loading.

The original CIANNA copyright and license headers are retained in the source
files. ORSDet-specific attribution is recorded in the repository-level
`NOTICE.md`.
